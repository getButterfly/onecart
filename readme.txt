== Description ==

One Cart is a responsive eCommerce WordPress theme.

For more information about OneCart please go to https://getbutterfly.com/downloads/onecart/.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Browse to your OneCart.zip theme package, upload it and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://getbutterfly.com/support/documentation/onecart-documentation/ for a guide on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
